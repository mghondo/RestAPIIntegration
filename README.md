# REST API Integration

This is a Salesforce project for REST API integration.

## Project Structure

- `force-app/` - Contains the Salesforce application code
- `sfdx-project.json` - Salesforce DX project configuration

## Getting Started

1. Make sure you have the Salesforce CLI installed
2. Authorize your Salesforce org
3. Deploy the code to your org

## Usage

This project provides REST API integration capabilities for Salesforce.
